echess — Electromagnetic Chessboard

2026-04-17 · git · https://github.com/jsharf/echess

Electromagnetic chessboard: a board with magnetic pieces moved by a grid of PCB coils
underneath.

BACKGROUND
The initial plan was to use off-the-shelf inductors under each square. That didn't work —
the ferrous cores attracted the magnets in the pieces directly, overwhelming the weak
field of the energized coil. After Carl Bugeja's PCB-coils video, the project pivoted to
coreless planar coils etched directly into a 4-layer PCB. No ferrous core means no
parasitic attraction.

LAYOUT
- pcb_coils/coil_gen.py — 4-layer square-spiral coil generator (.kicad_mod + preview PDF).
- pcb_coils/test_coil_visual.py — occupancy-grid router for visual validation.
- field.py — FEniCS magnetostatics simulation (WIP).
- BOM.md — parts list for inductor and control boards.

CURRENT BOARD
12.7mm square coils, 7 turns/layer, 0.5mm trace, 0.1mm clearance. 5x5 variant uses
10.16mm coils on the same 50.8mm board. JLCPCB Gerbers + drill files generated.
