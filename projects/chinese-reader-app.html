Chinese Reader App

2026-05-01 · mtime

A reader for studying Chinese. Long-press a word to get pinyin + a translation;
TTS reads the passage with the current word underlined as it's spoken; a content
pipeline scrapes and re-grades source material so you can read at the right level.

Started as a Flutter app, now pivoting to Expo / React Native to share more of
the content-pipeline code with the desktop tooling.

Recent work areas:
- Custom swipe physics so page-turn vs. long-press scrubbing don't fight.
- TTS progress callbacks driving in-line word highlighting.
- Pinyin rendering above each character with sentence-level yellow highlight.
- Re-grading pipeline for source content so passages match the reader's level.
